# Espace étudiant Academy

L’accès à `/academy/cours/[slug]/apprendre` nécessite un étudiant connecté et un enrollment actif ou terminé non expiré.

Le lecteur vidéo ne charge les vidéos Cloudflare qu’après vérification d’accès via une route API sécurisée. Les étudiants ne voient pas d’URL vidéo en clair dans l’interface et aucun bouton de téléchargement n’est affiché.
