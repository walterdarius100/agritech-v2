# Navigation publique

## Academy

La navigation publique met en avant `Academy` avec un lien vers `/academy` dans la navbar desktop, le menu mobile et les liens d’exploration du footer.

La page d’accueil présente désormais `Agri-tech Academy` comme l’espace de formation d’Agri-tech, avec le CTA `Découvrir notre Academy` vers `/academy`.

## Anciennes pages formations

Les anciennes routes `/formations` peuvent rester disponibles pour compatibilité, mais les liens publics principaux doivent orienter vers l’Academy afin d’éviter la confusion entre `Formations` et `Academy`.
